# Transforms

Transforms are helper functions you can pass to the `DataView` columns which will tweak the data in common ways.

You can pass multiple transforms to chain them together.

```js
import { DataView } from "@simpleview/sv-mosaic";
function myComponent(props) {
    const config = [
        ...
        columns : [
            {
                name : "test",
                label : "Test",
                transforms : [
                    TRANSFORMS
                ]
            }
        ]
    ]

    return (
        <DataView
            {...config}
        />
    )
}
```
